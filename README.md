# Next React App Demo

This is a demo of a dummy web-app built on top of the following technologies:

1. React (Hooks, Portals, etc.)
2. NextJS
3. Typescript
4. Material-UI
5. React-Hook-Form + Yup

## Installation

Refer to NextJS Getting Started [Docs](https://nextjs.org/docs)
